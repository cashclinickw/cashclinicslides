# كاش كلينيك – النظام المتكامل

نظام موحّد لكاش كلينيك يجمع ٣ أجزاء قابلة للنشر على GitHub Pages:

1. **الموقع التعريفي** (`index.html`) – للجمهور العام والجمعيات المحتمَلة
2. **العرض التقديمي** (`deck/`) – Pitch Deck بـ٢٠ شريحة احترافية
3. **نظام التقارير الداخلي** (`dashboard.html` + `reports/`) – للفريق

---

## 🎯 العرض التقديمي (Pitch Deck)

`deck/index.html` – ٢٠ شريحة تشمل:

- نبذة عن كاش كلينيك والرؤية
- الأرقام التراكمية (٤٬٢٣١ جلسة في ٦ دول، ٣ سنوات)
- التوزيع الجغرافي ومسار النمو
- المشكلة والحل
- منهجية الفنل المقلوب (٥ مستويات)
- القسمين (العائلة + رواد الأعمال) والفريق
- آلية الشراكة الكاملة مع الجمعيات
- نظام التقارير (١٣ وثيقة)
- التسعير المدعوم للجمعيات
- معلومات التواصل

### اختصارات لوحة المفاتيح:
| المفتاح | الإجراء |
|---------|---------|
| → / ↑ / Page Up | السابق |
| ← / ↓ / Page Down / Space | التالي |
| Home / End | الأولى / الأخيرة |
| F | ملء الشاشة |
| P | طباعة PDF |
| O / Esc | عرض شامل لكل الشرائح |

يدعم أيضًا السحب على الموبايل (swipe) والتنقل عبر URL hash (`#5` للشريحة الخامسة).

## 🌐 الموقع التعريفي

`index.html` – صفحة هبوط احترافية تشمل:
- المشكلة والحل
- الأقسام والخدمات
- منهجية الـ٤ جلسات
- آلية الشراكة مع الجمعيات (٧ خطوات)
- الفريق
- روابط للعرض التقديمي ونظام التقارير

---

## 📊 نظام التقارير الداخلي

`dashboard.html` ← اللوحة الرئيسية لـ١٣ وثيقة:

**للمستفيد (٥):** إقرار + ٤ كرّاسات للجلسات
**للجمعية (٧):** نموذج استلام + نموذج قرار + ٤ تقارير جلسات + التقرير الشامل
**المالي (١):** فاتورة تسوية بعد كل جلسة

كل وثيقة:
- حفظ تلقائي في المتصفح (localStorage)
- تصدير/استيراد JSON
- طباعة PDF احترافي A4
- عربي RTL كامل بهوية كاش كلينيك

## 🚀 النشر على GitHub Pages

١. أنشئ حساب GitHub (إذا ما عندك): https://github.com/signup
٢. أنشئ Repository جديد (Public أو Private)
٣. ارفع كل محتويات هذا المجلد عبر سحب/إفلات
٤. `Settings` → `Pages` → Source: `main` / `(root)` → `Save`
٥. انتظر دقيقتين – رح يطلع رابط: `https://<username>.github.io/<repo-name>/`

### الروابط بعد النشر:
- `/` – الموقع التعريفي
- `/deck/` – العرض التقديمي
- `/dashboard.html` – لوحة التقارير
- `/reports/<اسم-التقرير>.html` – أي تقرير فردي

## 🔒 الخصوصية

- بيانات التقارير تُحفظ **محليًا في المتصفح فقط** (localStorage)
- **لا تُرسل لأي خادم**، ولا حتى لـGitHub

- لمزامنة بين أجهزة: استخدم زر «تصدير JSON» ثم «استيراد»
